# 🛡️ SentinelLLM Frontend

AI-powered dashboard to detect sensitive data leakage in LLM prompts.

## 🚀 Features
- Real-time prompt analysis
- Risk scoring system
- Detection of:
  - API keys
  - Emails
  - Names
- Clean UI dashboard

## 🔗 Backend API
https://sentinelllm-ai-data-leakage.onrender.com

## 🧠 Tech Stack
- React
- Axios
- FastAPI (backend)

## 🎯 Use Case
Helps companies prevent accidental data leaks while using AI tools.
